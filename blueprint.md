# Korean Lotto Number Generator

## Overview

This project is a web application that explains the Korean lottery and generates 10 sets of lottery numbers for the user.

## Project Outline

### Style and Design
*   **Layout:** A clean and simple layout with a clear information section and a prominent number generation section.
*   **Color Palette:** A modern and visually appealing color scheme.
*   **Typography:** Clear and readable fonts.
*   **Interactivity:** A button with a clear call-to-action to generate the numbers.

### Features
*   An explanation of the Korean lottery system.
*   A button to generate 10 unique sets of lottery numbers.
*   A display area for the generated numbers.

## Current Request

**Request:** Create a program that explains the Korean lottery and shows 10 sets of lottery numbers.

**Plan:**
1.  **Explain the Korean Lotto:** Provide a brief explanation of the Korean lottery system in the HTML.
2.  **Create the UI:**
    *   An information section to explain the lottery.
    *   A button to generate lottery numbers.
    *   A section to display the generated numbers.
3.  **Style the application:** Apply CSS for a visually appealing design.
4.  **Implement the logic:**
    *   Generate 10 unique sets of 6 random numbers between 1 and 45.
    *   Display the generated numbers when the button is clicked.
